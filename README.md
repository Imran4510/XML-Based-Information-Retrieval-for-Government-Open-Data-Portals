
# XML-Based Crop Data Retrieval System

## Description
This project is an XML-based crop data retrieval system developed as part of an ADBMS mini project. It allows users to search and retrieve agricultural crop data using a simple user interface. The system works with structured crop data and helps users access relevant information efficiently.

## Features
- Retrieve crop data from XML files
- Simple and user-friendly interface
- Structured agricultural data handling
- Easy data inspection and retrieval
- Supports data processing and analysis

## Technologies Used
- Python
- Streamlit
- XML
- CSV
- Jupyter Notebook

## Project Objective
The main objective of this project is to demonstrate how XML can be used for storing, managing, and retrieving structured agricultural data efficiently through a user-friendly application.

## Files Included
- `app.py` – Main application file
- `data.xml` – XML dataset used for retrieval
- `ICRISAT-District Level Data.csv` – Source agricultural dataset
- `inspect_xml.py` – Script to inspect and analyze XML data
- `Untitled.ipynb` – Notebook used for testing or data processing

## How to Run
1. Clone the repository:
   git clone <your-repository-link>

2. Open the project folder:
   cd <project-folder-name>

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   streamlit run app.py
