# XML-Based Crop Data Retrieval System

## Description
This project is an XML-based crop data retrieval system developed as part of an ADBMS mini project. It allows users to search and retrieve agricultural crop data using a simple user interface by selecting State, Year, and Crop.

## Features
- Retrieve crop data from XML files
- Search using filters like State, Year, and Crop
- Simple and user-friendly interface
- Fast and structured data retrieval

## Technologies Used
- Python
- Streamlit
- XML
- XPath / ElementTree

## Project Objective
The main objective of this project is to demonstrate how XML can be used for storing and retrieving structured agricultural data efficiently through a user-friendly application.

## How to Run
1. Clone the repository:
   git clone <your-repository-link>

2. Open the project folder:
   cd <project-folder-name>

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   streamlit run app.py
